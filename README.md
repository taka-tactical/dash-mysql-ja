dash-mysql
============

Generates a [Dash](http://kapeli.com/dash) docset for [MySQL](http://www.mysql.com).

Support Only [5.6 Japanese document](https://dev.mysql.com/doc/refman/5.6/ja/)

usage
============

cd "checkout dir"

php generate-mysql.php
